# 🛒 E-commerce Funnel Analysis

## 📌 Project Overview
This project analyzes user behavior in an e-commerce platform to identify where customers drop off in the purchase journey. The goal is to improve conversion rates by understanding the sales funnel.

---

## 🎯 Problem Statement
Many users visit an e-commerce website, but only a few complete purchases.  
This project aims to identify **at which stage users drop off** in the funnel:
**Visit → Add to Cart → Checkout → Purchase**

---

## 🧰 Tools & Technologies Used
- Python (Pandas) – Data Cleaning
- SQL (MySQL) – Data Analysis
- Power BI – Data Visualization

---

## 📂 Dataset
The dataset contains user activity with the following columns:
- user_id
- event (visit, add_to_cart, checkout, purchase)
- timestamp
- product
- price

---

## ⚙️ Steps Performed

### 1. Data Cleaning (Python)
- Removed duplicates  
- Converted timestamp format  
- Handled missing values  

### 2. SQL Analysis
- Counted users at each stage of the funnel  
- Calculated conversion rates  
- Identified drop-off points  

### 3. Data Visualization (Power BI)
- Created Funnel Chart  
- Added KPI cards (Total Users, Revenue)  
- Built interactive dashboard  

---

## 📊 Key Insights
- 75% of users added items to cart  
- 66% proceeded to checkout  
- Only 50% completed the purchase  

👉 **Major drop occurs at Checkout → Purchase stage**

---

## 💡 Business Recommendations
- Simplify checkout process  
- Improve payment experience  
- Offer discounts or incentives
